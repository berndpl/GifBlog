:: ANIMATION FLOOR 0.1 - gallery php script

WHAT IS ANIMATION FLOOR?
it's kind of a photo-blog system for animated gifs. one page, showing a bunch of GIFs in a chronological order with title-overlays. that's it.

WHERE DO I GET IT?
just write an email an ask for the script and i will send it to you. i thought about releasing it unter GPL and setup a SVN or something but i didn't deal with it yet. so if you have any use for this just tell me: bernd@plontsch.de

HOW DO YOU MAKE THESE ANIMATIONS?
all photos are taken by my iphone and the amazing quadcamera app. some photos are converted to simple animated GIFs with the quadanimator, great stuff! but of course you can use any other GIF as well. right now i am using my QUAD2GIF (look down for a description and download) batch script to convert a lot of quadcamera images the quick way.

HOW DOES IT WORK?
in order to post a photo, you just upload it to your webspace (where the ANIMATION FLOOR script lives) and .... well, that's it. there is no backend, database or cms. just PHP and IMAGEMAGICK. the only thing to remember is to name your files according to the scheme: "090813_nicetitle.gif" ... this will put an image with the timemark "13.08.09" and the title "nicetitle" online. if you don't want any title use the keyword "blank", e.g. "090813_blank.gif". there is even a converterscript for new photos, so you don't have to neccessarily use the quadanimator.

WHY DON'T YOU JUST USE SYSTEM XY? IT'S WAY COOLER!
either because i didn't know that it exists or because i thought that it is too bloated for my purpose. when i was looking for a slick and easy photo-blogging or gallery script treating my gif-animated lomo photos right, i couldn't find any! ... so i had to do it by myself :). but tell me about it!

FEATURES
* handles (only) animated gif
* no cms, just upload images to publish
* metadata (date + title) parsed from image filenames
* converter for lomostyle quadimages to animated gifs

:: QUAD2GIF 0.1 - unix batch script for bulk conversions of quadcamera images to animated gifs

WHAT IS c?
it's a simple unix batch script to convert folders full of quadcamera images into animated gifs with the press of one button. to run it you need imagemagick installed on your system, it will do the imageprocessing part. on mac os x you can get it with the darwin ports (it's a repository of unix ports for mac os x. great stuff.). if you feel ready just download the script here, put it into the folder with you quadcamera images and run it by typing "./quad2gif" in the terminal. have fun!
